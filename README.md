#### hexo
1. 安装hexo
```
npm install hexo-cli -g
```

2. 检查版本
```
hexo v 
```

3. 初始化
```
hexo init
```

4. 安装依赖
```
npm install
```

5. 终端运行
```
hexo s
```

安装主题渲染
```
$ git clone https://github.com/chaooo/hexo-theme-BlueLake.git themes/BlueLake
$ npm install hexo-renderer-jade@0.3.0 --save
$ npm install hexo-renderer-stylus --save
```
